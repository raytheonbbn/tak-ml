1. Make sure TAK Server is installed and running (currently supports 4.10).
    Note, make sure to target at least version takserver-docker-4.10-RELEASE-72
    (can be downloaded from https://artifacts.tak.gov/ui/repos/tree/General/TAKServer/release/4.10-RELEASE-72/takserver-docker-4.10-RELEASE-72.zip)
2. Install Postgresql (e.g. on Ubuntu: sudo apt install postgresql postgresql-contrib)
3. Update application.properties and point to correct TAK Certs
4. run run.sh
